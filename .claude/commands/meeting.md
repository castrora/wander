Capture and file meeting notes for today's Wander session.

Ask the user (or infer from conversation context) for:
- Meeting type (working session / decision session / research review / other)
- Key topics discussed
- Decisions made (to be added to decision-log.md)
- New risks or actions identified (to be added to risk-log.md)

Then:
1. Create a meeting notes file at pm/meetings/YYYY-MM-DD-[topic].md using today's date
2. Append any new decisions to pm/decisions/decision-log.md with the next D-### number
3. Append any new risks/actions to pm/risks/risk-log.md with the next R-### / A-### number
4. Summarize what was filed in the conversation

Format for meeting notes:
---
# [Meeting Type] — [Topic]
**Date:** YYYY-MM-DD
**Attendees:** Ray Castro, Claude

## Purpose
[one sentence]

## Key Discussion Points
[bullet points]

## Decisions Made
[link to decision log entries]

## Open Questions / Action Items
[link to risk log entries]

## Next Session Focus
[top 3 priorities for next session]
---
